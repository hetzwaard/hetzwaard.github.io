README Notes

This page runs the C ray-casting engine compiled with Emscripten.

Ray Casting Logic (C)
  1. For each screen column, build a ray from player direction and camera plane.
  2. Run DDA on the map grid until the ray hits a wall tile.
  3. Use perpendicular distance to compute wall slice height (no fish-eye).
  4. Sample texture coordinates and draw one vertical stripe.
  5. Draw floor and ceiling and apply distance-based shading.
// simplified render pass
for (int x = 0; x < render_w; ++x) {
    ray_setup(&ray, player, camera, x);
    dda_cast(&ray, map);
    wall_project(&ray, render_h);
    wall_draw_stripe(&ray, x);
    floor_draw_column(&ray, x);
}
Bonus Logic Used Here
  • Bonus mode is always enabled for this web build.
  • Map is fixed to /maps/default.cub.
  • Keys, doors, minimap, sprites, lava, and checkpoints are active.
